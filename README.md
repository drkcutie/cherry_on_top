## üóíÔ∏è Project Overview

RESIKLO is an AI-driven waste reduction and tracking platform designed to make sustainable living effortless and engaging. Using an AI-powered waste scanner, users can upload photos of waste, and the system will identify the best disposal method based on type and confidence level, using YOLO for object detection. Users can track their personal waste reduction score over time, compete on leaderboards, and participate in social challenges, much like Strava. Additionally, Resiklo AI can partner with organizations and businesses to offer rewards‚Äîsuch as vouchers or discounts‚Äîin exchange for waste contributions, like collecting 50 used 500ml water bottles for exclusive perks, making sustainability more rewarding.

### Functional Requirements

üîë The system must accept images taken from the camera directly.

üîë The system must calculate a waste reduction score based on waste type, confidence level, and unit count.

üîë The leaderboard must update in real-time based on users‚Äô waste contributions.

üîë The rewards system must track waste submissions and integrate with partnered businesses for voucher distribution.

## üõ†Ô∏è Installation Guide

> **Prerequisite:** Node.js v22 or higher

### Steps:

1. Clone the repository:

   ```sh
   git clone https://github.com/drkcutie/cherry_on_top.git
   cd cherry_on_top
   ```

2. Install frontend dependencies:

   ```sh
   cd resiklo
   npm install
   ```

3. Run expo on preferred device

   - For Android

     ```sh
     npm run android
     ```

   - For IOS

     ```sh
     npm run ios
     ```
